============================================================
  LeLab-zh 离线安装说明
============================================================

本安装包为完全离线版本，安装过程不需要网络连接。

【系统要求】

Windows 版：
  - Windows 10/11 64 位
  - C 盘可用空间 >= 30GB（推荐 40GB）
  - 有 NVIDIA 显卡可启用 CUDA 加速（无则 CPU 模式，训练较慢）
  - 需要管理员权限安装 USB 串口驱动（LeLab 本身不需要）

macOS 版：
  - Apple Silicon Mac（M1/M2/M3/M4）
  - 可用空间 >= 8GB（推荐 12GB）
  - 不支持 Intel Mac

【安装步骤】

1. 将压缩包复制到电脑本地（不要直接在压缩包内运行）
2. 右键完整解压
3. 双击「一键安装」
4. 等待安装完成（首次安装可能需要几分钟）
5. 双击桌面「启动LeLab」

【常见问题】

Q: 提示"安装文件不完整"？
A: 请先将压缩包完整解压到本地文件夹，再运行安装脚本。
   禁止在压缩包预览界面直接双击。

Q: macOS 提示"无法验证开发者"？
A: 前往「系统设置 -> 隐私与安全性」，找到 LeLab-zh 相关提示，
   点击「仍要打开」。

Q: Windows 提示"无法运行"？
A: 右键安装脚本 -> 属性 -> 勾选「解除锁定」-> 确定，再重新运行。

Q: 安装后浏览器没有自动打开？
A: 手动访问 http://127.0.0.1:8000

Q: 卸载会删除我的数据吗？
A: 不会。卸载只删除程序文件和桌面快捷方式。
   用户数据集、HF 缓存、模型文件、训练结果均保留。

【离线功能】

可用：校准、遥操作、录制、回放、本地训练（CPU/GPU）
不可用：模型权重下载、Hub 登录、云端数据集、云端训练
