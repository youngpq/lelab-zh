============================================================
  LeLab-zh 离线安装说明
============================================================

本安装包为完全离线版本，安装过程不需要网络连接。

【系统要求】

Windows 版：
  - Windows 10/11 64 位
  - 安装目标磁盘可用空间 >= 30GB（推荐 40GB）
  - 有 NVIDIA 显卡可启用 CUDA 加速（无则 CPU 模式，训练较慢）
  - 需要管理员权限安装 USB 串口驱动（LeLab 本身不需要）

macOS 版：
  - Apple Silicon Mac（M1/M2/M3/M4）
  - 可用空间 >= 8GB（推荐 12GB）
  - 不支持 Intel Mac

【安装步骤】

1. 将压缩包复制到电脑本地（不要直接在压缩包内运行）
2. 右键完整解压
3. Windows 双击 `windows_install.bat`；macOS 双击 `macos_install.command`
4. 出现安装位置提示时，直接按回车使用默认位置；如 C 盘空间不足，可输入例如 D:\LeLab-zh
5. 等待安装完成（首次安装可能需要几分钟）
6. 双击桌面「启动LeLab」

【macOS 首次运行方法】

如果 macOS 双击 `macos_install.command` 提示无法打开，先进入已经解压好的安装文件夹：

1. 在 Finder 中右键该文件夹，选择「服务 -> 新建位于文件夹位置的终端窗口」或「在终端中打开」；
2. 在打开的终端中复制并执行下面三行：

   chmod +x ./*.command ./uv/uv ./runtime/bin/python3.12
   xattr -dr com.apple.quarantine .
   ./macos_install.command

如果右键菜单没有「在终端中打开」，先打开“终端”应用，再把解压后的安装文件夹拖进终端窗口，按回车进入该目录，然后执行上面三行。

这些命令只处理当前离线安装包的执行权限和 macOS 隔离标记，不会访问网络，也不会修改其他文件。

【常见问题】

Q: 提示"安装文件不完整"？
A: 请先将压缩包完整解压到本地文件夹，再运行安装脚本。
   禁止在压缩包预览界面直接双击。

Q: macOS 提示"无法验证开发者"或双击没有反应？
A: 请先完整解压，再按本说明中的「macOS 首次运行方法」执行三行命令。
   也可以在 Finder 中按住 Control 点击 `macos_install.command`，选择「打开」，
   再在「系统设置 -> 隐私与安全性」中点击「仍要打开」。

Q: Windows 提示"无法运行"？
A: 右键安装脚本 -> 属性 -> 勾选「解除锁定」-> 确定，再重新运行。

Q: 全新 Windows 电脑需要预装 VC++、Python 或 uv 吗？
A: 不需要。一键安装会自动运行包内的 prerequisites\VC_redist.x64.exe；Python 和 uv 也都包含在包内。
   如果 Windows 弹出 UAC，请选择“是”；如提示需要重启，请重启后再次运行一键安装。

Q: PowerShell 中输入 lelab-zh 或 uv 找不到命令怎么办？
A: 这是正常的，安装包不修改系统 PATH。请使用桌面快捷方式启动；排错时使用安装目录中的绝对路径。

Q: 安装后浏览器没有自动打开？
A: 手动访问 http://127.0.0.1:8000

Q: 可以安装到 D 盘或其他位置吗？
A: 可以。安装时输入本机固定磁盘上的完整路径，例如 D:\LeLab-zh。
   请勿选择 U 盘、网络盘或当前解压出的安装包文件夹。

Q: 卸载会删除我的数据吗？
A: 不会。卸载只删除程序文件和桌面快捷方式。
   用户数据集、HF 缓存、模型文件、训练结果均保留。

【离线功能】

可用：校准、遥操作、录制、回放、本地训练（CPU/GPU）
不可用：模型权重下载、Hub 登录、云端数据集、云端训练
