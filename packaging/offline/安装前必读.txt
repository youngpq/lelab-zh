============================================================
  安装前必读 - 硬件与系统检查
============================================================

本安装包安装 LeLab-zh 本体不需要管理员权限。
但以下硬件驱动可能需要管理员权限安装，请在培训前完成。

============================================================
  Windows 检查卡
============================================================

[ ] 仅限 Windows 64 位
[ ] C 盘可用空间 >= 40GB
[ ] 有 NVIDIA 显卡则启用 CUDA 加速（无则 CPU 模式，训练较慢）
[ ] 已识别 SO-101 串口设备

需要管理员权限的操作：
  - USB 串口驱动安装（见下方驱动说明）
  - VC++ 2015-2022 x64 运行库安装
  - NVIDIA 驱动更新（如有需要）

============================================================
  macOS 检查卡
============================================================

[ ] 仅限 Apple Silicon Mac（M1/M2/M3/M4）
[ ] 可用空间 >= 8GB（推荐 12GB）
[ ] 已识别 SO-101 串口设备

首次运行需放行 Gatekeeper：
  系统设置 -> 隐私与安全性 -> 找到 LeLab-zh 提示 -> 仍要打开

============================================================
  USB 串口驱动
============================================================

SO-101 控制板使用 USB 转串口芯片，需安装对应驱动。

请根据实际控制板上的芯片型号选择：

Windows：
  - FTDI 芯片：https://ftdichip.com/drivers/vcp-drivers/
  - CH340 芯片：https://www.wch-ic.com/downloads/CH341SER_EXE.html
  - CP2102 芯片：https://www.silabs.com/developers/usb-to-uart-bridge-vcp-drivers

安装后检查：设备管理器 -> 端口(COM 和 LPT) -> 应显示 "USB Serial Port (COMx)"

macOS：
  - FTDI 驱动：https://ftdichip.com/drivers/vcp-drivers/
  - 安装后检查：终端运行 `ls /dev/cu.usb*` 应显示设备

============================================================
  VC++ 运行库（仅 Windows）
============================================================

LeLab 依赖 Microsoft Visual C++ 2015-2022 x64 运行库。
一键安装会自动运行安装包附带的：
  prerequisites/VC_redist.x64.exe
如 Windows 弹出 UAC，请选择“是”。如果提示需要重启，请重启后再次运行一键安装。

============================================================
  相机权限（如使用手机摄像头）
============================================================

macOS：系统设置 -> 隐私与安全性 -> 相机 -> 允许浏览器访问
Windows：设置 -> 隐私 -> 相机 -> 允许应用访问相机
